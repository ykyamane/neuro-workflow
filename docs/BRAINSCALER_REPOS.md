# Related: OIST BrainScaler repos

The OIST repositories **brainscaler** and **brainscaler_frontend** are related to this project but **independent** of the main neuro-workflow code. They are installed inside this repo in a dedicated folder.

- **Location:** `brainscaler/` (inside neuro-workflow)
  - `brainscaler/brainscaler` – backend (knowledge graph, ontologies, RAG)
  - `brainscaler/brainscaler_frontend` – frontend (Conversational AI web app)
- **Repos:** [oist/brainscaler](https://github.com/oist/brainscaler), [oist/brainscaler_frontend](https://github.com/oist/brainscaler_frontend)

Setup and run instructions (using conda env `neuro`): see **`brainscaler/README.md`** in this repo.
